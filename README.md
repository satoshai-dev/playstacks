# Playstacks

Playstacks is an extension of Playwright that adds support for Stack Wallet extensions, enabling end-to-end (E2E) testing for applications utilizing these wallet features.

## Installation

To install Playstacks, you'll need to add it as an npm dependency in your project:

```bash
npm install playstacks
```
